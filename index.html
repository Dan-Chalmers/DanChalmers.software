AWS Virtual Assistant
Python 3.7

My aim for this project was to create my very own virtual assistant similar to Amazon's Alexa and Apple's Siri using AWS to host it on the cloud. The project uses APIs to access Reddit for facts, play the radio, make phone calls, send SMS messages and find out the weather. I used Request Handlers and AWS Lambda to allow the bot to respond. Each intent has multiple utterances.

Morse Code Reader
Python 3.5/SQL

In this project, I set out to make a device that could aid in the learning of Morse Code. I used a Raspberry Pi and a Custom-made interface board. The software can accurately time each button press and produce a translated, English string from the inputted Morse Code and vice versa. This project incorporated an external Leadreboard and an automatic emailing system.

Augmented Reality App
C#

This Android Application, made in the Unity Game Engine, displays a user's camera on their screen and overlays a fixed marker onto it. When the user taps the screen, a model plane I created will appear where the marker was. This plane will then be fixed in place and the user can move the camera around it on all axes as well as being able to increase and decrease the size of the model.

Client/Server Encryption
C++

As part of a larger project, I designed and implemented an Encryption Algorithm. It uses two public keys, a private key and a checksum to encode an inputted string. It would take 9^(2n-1) attempts to brutefore the encyption where n is the length of the string. This means that, the average sentence would take 2.95e+94 attempts to bruteforce.
